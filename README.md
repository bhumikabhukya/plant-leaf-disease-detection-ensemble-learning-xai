Developed a machine learning-based system to detect and classify plant leaf diseases from images using Ensemble Learning techniques. Integrated Explainable AI (SHAP/LIME) to provide transparent and interpretable predictions
# Plant Leaf Disease Detection Using Ensemble Learning and Explainable AI

## Project Overview
This project focuses on detecting and classifying plant leaf diseases using Ensemble Learning and Explainable AI (XAI). The system analyzes leaf images, predicts the disease category, and provides interpretable explanations for the prediction to support better agricultural decision-making.

## Features
- Plant leaf disease classification from images
- Ensemble Learning for improved prediction accuracy
- Explainable AI using SHAP and LIME
- Image preprocessing and feature extraction
- Performance evaluation using standard metrics

## Technologies Used
- Python
- Machine Learning
- Deep Learning (CNN)
- TensorFlow / Keras
- Scikit-learn
- OpenCV
- NumPy
- Pandas
- Matplotlib
- SHAP
- LIME

## Dataset
The project uses plant leaf image datasets containing healthy and diseased leaf samples. Images are preprocessed before training and testing the models.

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Image Enhancement
4. Model Training
5. Ensemble Learning
6. Disease Prediction
7. Explainable AI Analysis
8. Performance Evaluation
